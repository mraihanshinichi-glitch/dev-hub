# Project Templates

DevHub menyediakan template project untuk mempercepat setup dan development. Template ini mencakup notes panduan, features roadmap, dan release plan.

## Cara Menggunakan Template

1. **Buat Project Baru**: Klik "Buat Project" di dashboard
2. **Pilih Template**: Pilih dari template yang tersedia atau buat project custom
3. **Preview Template**: Lihat detail notes, features, dan releases yang akan dibuat
4. **Customize**: Sesuaikan nama dan deskripsi project
5. **Create**: Template akan otomatis membuat semua content

## Fitur Template

### Notes Panduan
- Setup instructions dan best practices
- Tech stack recommendations
- Development workflow guidelines
- Architecture dan design patterns

### Features Roadmap
- Pre-defined features sesuai jenis project
- Organized by categories (setup, feature, enhancement, etc.)
- Status "planned" untuk semua features
- Detailed descriptions untuk setiap feature

### Release Plan
- Version planning (0.1.0, 0.2.0, 1.0.0)
- Release notes dan milestones
- Category-based releases (major, minor, patch)

## Customization

Setelah project dibuat dari template:

1. **Edit Notes**: Sesuaikan panduan dengan kebutuhan spesifik
2. **Modify Features**: Tambah, edit, atau hapus features
3. **Update Releases**: Sesuaikan version dan timeline
4. **Add Categories**: Buat kategori custom di Settings

## Project Custom

Jika template tidak sesuai kebutuhan:

1. Pilih "Project Custom" 
2. Mulai dengan project kosong
3. Tambah notes, features, dan releases manual
4. Gunakan AI Assistant untuk ide dan saran

## Tips Menggunakan Template

- **Review semua content**: Periksa notes dan features yang dibuat
- **Sesuaikan dengan kebutuhan**: Edit atau hapus yang tidak relevan  
- **Gunakan AI Assistant**: Minta saran untuk improve template
- **Update secara berkala**: Sesuaikan roadmap seiring development
- **Share dengan tim**: Template bisa jadi reference untuk tim development
